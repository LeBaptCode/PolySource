## Répertoire Exécutables

Ce répertoire contient les exécutables pour les deux modules GSM utilisés dans ce projet, WH-LTE-7S1 et USR-DR154. Ces exécutables sont utilisés pour configurer et gérer les modules GSM selon leurs fiches techniques et manuels utilisateur respectifs. Les exécutables sont conçus pour faciliter la communication avec les modules GSM, permettant des tâches telles que l'envoi de commandes AT, la gestion des connexions réseau et le traitement de la transmission de données.

### Qu'y a-t-il dans le répertoire Exécutables ?

Dans le répertoire Exécutables, vous trouverez les fichiers suivants :
- `Module.exe` : Cet exécutable est spécifiquement conçu pour le module GSM WH-LTE-7S1. Il fournit une interface conviviale pour configurer et gérer le module, incluant des fonctionnalités telles que l'enregistrement réseau et la surveillance de la force du signal.
- `Cfg.ini` : Ce fichier de configuration contient les paramètres pour les exécutables, permettant aux utilisateurs de personnaliser leur expérience et d'assurer la compatibilité avec leur module GSM spécifique.
- `Lang.txt` : Ce fichier contient les paramètres de langue pour les exécutables, permettant aux utilisateurs de sélectionner leur langue préférée pour l'interface et les messages affichés par les exécutables.
